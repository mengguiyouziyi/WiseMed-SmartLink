# 方言/翻译热词策略

## 1. 热词来源
- 医疗术语：ICD-10、基药目录、常见药物、检验指标。
- 方言词汇：衡水/保定/石家庄地方词；与医生/患者访谈收集。
- 政策词：医共体、医保控费、多方合作等。
- 流程词：低置信度、转人工、随访。

## 2. 维护流程
1. 术语收集：算法+医学顾问+客户成功每月更新，使用 CSV/词典格式。
2. 审核：医学顾问确认翻译/解释；语言专家确认方言拼写/发音。
3. 部署：
   - ASR：加载热词词典（WFST/N-gram）；ESPnet/WeNet `hotword` 配置。
   - NMT：Glossary Constraint + 术语替换；提供术语优先列表。
4. 监控：统计热词命中率、遗漏；低置信度样本进入人工复核。

## 3. 词典格式
```
term,category,standard,variant,dialect,priority
肺结节,影像,肺结节,肺疙瘩,肺疙瘩,high
...
```
- `priority`: high/medium/low
- `variant`: 允许多种读法

## 4. 部署策略
- **ASR**：
  - 在解码时增加热词权重（如 WeNet `--hotword-weight`）。
  - 热词长度控制在 1-5 个汉字；超长术语拆分。
  - 方言读音使用拼音/音标辅助。
- **NMT**：
  - 使用 `lexically constrained decoding`；将术语映射提供给翻译引擎。
  - 翻译后进行术语对齐校验；偏差自动纠正。
- **对话**：Rasa Slot/NLU 加入热词特征；意图检测中放大权重。

## 5. 生命周期
1. `collect` → `review` → `deploy` → `monitor` → `retire`
2. 老旧或错误热词需归档，避免污染模型。
3. 每季度导出热词版本，配套 Model Card 中记录。

## 6. 风险控制
- 防止热词过拟合：测试常规目标以避免误识别。
- 热词更新需经过 QA；高优热词上线需双人校验。
- 记录热词变更，供合规审计。
