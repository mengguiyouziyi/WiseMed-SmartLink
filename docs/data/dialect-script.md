# 方言采集剧本（衡水/保定）

## 1. 结构
- 场景 A：门诊问诊（普通话 + 方言混合）
- 场景 B：慢病随访
- 场景 C：用药宣教 / 安全提示
- 场景 D：涉外患者翻译（中→英、英→中）
- 场景 E：应急与禁忌确认

## 2. 剧本示例
### 场景 A：门诊问诊
- 医生（衡水方言）：
  - “今儿哪里不舒坦？”
  - “咳得厉害不？有痰没？”
  - “发烧多高？吃过药没？”
- 患者：
  - “连着咳嗽两礼拜，晚上更厉害。”
  - “吃过自个买的止咳药，没啥用。”
- 目标：捕捉主诉、病程、用药史、过敏史。

### 场景 B：慢病随访（高血压 + 糖尿病）
- 客服（普通话+方言）：
  - “血压计还按时测不？最近测的数是啥？”
  - “血糖饭前饭后分别是多少？”
  - “药没落下吧？有没有啥不舒服？”
- 患者：
  - “血压早晨 140/90，晚上差不多。”
  - “血糖饭后老高，糖口有点麻。”
- 目标：收集指标、用药遵从性、症状。

### 场景 C：用药宣教
- 药师：
  - “这瓶胰岛素放冰箱冷藏，不能冻。”
  - “照着医生开的剂量，早晚各打一次。”
  - “要是头晕或出汗赶紧吃糖再联系我们。”
- 患者：复述确认。

### 场景 D：涉外患者翻译
- 患者（英文）：
  - “I feel chest tightness and shortness of breath for two days.”
- 翻译（中文）：
  - “患者说有胸闷，气短，持续两天。”
- 医生 → 翻译 → 患者：
  - 重点在病史、过敏、宗教禁忌。

### 场景 E：禁忌确认
- “你有没有对药犯（过敏）？”
- “是不是怀孩子/备孕？”
- “宗教上有没有饮食禁忌？”

## 3. 录制要求
- 每段 2-3 分钟；包含问答、确认、情绪变化。
- 双通道收音：医生/患者分轨。
- 标记 meta：`{scene, role, dialect, emotion, keywords}`。
- 记录术语：使用 `medical-terms.csv` 交叉引用。

## 4. 扩展
- 计划每月新增 5 套剧本；纳入慢病、影像解释、手术同意等。
- 多角色：医生、护士、药师、翻译、患者、家属。
