# 合规体系与试点推进 SOP 草稿

## 1. SaMD 合规体系蓝图
- **QMS**：对齐 ISO 13485；核心流程（设计控制、供应商管理、CAPA、文档管理）配置在 eQMS（Notion/Confluence + Git 仓库）。
- **软件生命周期 (IEC 62304)**：
  - 等级：B（涉及临床决策但有医生复核）。
  - 产物：URS、SRS、架构说明、SOUP 清单、风险矩阵、SVVP、缺陷跟踪、配置管理记录。
- **风险管理 (ISO 14971)**：建立 Hazard 目录（诊断误判、翻译错误、数据泄露、模型漂移）；FMEA + 余量控制。
- **可用性 (IEC 62366)**：医生工作站可用性研究、对话翻译安全提示、低置信度交互设计。
- **网络安全**：
  - SBOM（Syft）、漏洞扫描（Grype/Trivy），CVSS ≥7 需 CAPA。
  - 零信任：OIDC + MFA，RBAC 与最小权限；传输加密（mTLS）、数据静态加密。
- **隐私与数据**：DPA、数据最小化、脱敏策略、联邦学习 + 差分隐私；访问审计 7 年留存。

| 文档 | 内容 | 所属标准 |

| 网络安全报告 | 威胁建模、SBOM、渗透测试 | NMPA 指南 |

## 3. 试点推进 SOP
### 阶段 0：机会评估（T-8 周）
1. 客户成功+政府关系获客 → 填写《需求问卷》（机构类型、影像量、现有 HIS/PACS）。
2. 技术评估会议：确认接口、安规、网络、合规要求。
3. 输出《试点立项书》：范围、KPI、资源、时间线。

### 阶段 1：实施准备（T-6 周）
1. 签署 DPA / SOW / 合作协议；明确责任边界。
2. 安排现场踏勘：机房、电源、网络、终端数量。
3. 输出《部署方案》：硬件清单、网络拓扑、项目日程。
4. 安排数据采样与对接：
   - 影像：选取最近 1 个月病例，进行 DICOM 兼容测试。
   - EMR/HIS：确定 HL7/FHIR 字段映射。
   - 翻译：采集方言样本，校对术语表。

### 阶段 2：部署与调试（T-4 周）
1. 上机安装：Orthanc、边缘设备、SaaS 适配包、ASR 终端。
2. 接口联调：C-STORE/C-FIND、HL7 ORU、FHIR DiagnosticReport、Kafka 事件。
3. 模型验证：
   - 影像：使用机构历史数据进行再验证（≥ 100 例），记录敏感度/特异度。
   - 翻译：WER/术语准确率；低置信度回退流程演练。
4. 安全：等保三级检查清单、渗透测试、SBOM 提交。

### 阶段 3：试运行（T-2 周）
1. 双轨运行：AI 结果仅供参考，与传统流程并行。
2. 培训：医生读片、护士随访、药师用药校验、管理员安全；不少于 2 场/角色。
3. 指标观测：
   - 影像周转时长、误诊率、推理延迟。
   - 翻译沟通时长、低置信度回退率。
   - SaaS 指标：库存预警准确率、处方合规率。
4. 收集反馈 → BUG/需求进入 Jira/禅道，优先级 P1/P2/P3。

### 阶段 4：验收与转商用（T0）
1. 验收测试：按《上线验收表》逐项核对。
2. 签发验收报告，进入正式计费；更新资产台账。
3. 安排在役监测：SLA、模型漂移检测、PMS 报告模板。

## 4. KPI & 报告节奏
- 周报：实施进度、阻塞风险、技术问题、合规项。
- 月度：试点KPI（第9节指标）、财务、客户满意度（NPS）。
- CAPA：合规/安全/临床相关事件 48h 内上报，根因分析 5 个工作日，关闭时长 <30 天。

## 5. 政策与资金对接流程
1. 设立“政策雷达”：跟踪千县工程、协同发展基金、数字医疗专项；在 Notion 建台账。
2. 申请步骤：
   - 依据项目阶段匹配资金（如试点阶段申请智慧医疗专项）。
   - 准备材料：商业计划书、技术白皮书、合规证明、预算表。
   - 政府关系经理对接卫健委/科技厅，安排专家评审。
3. 资金使用：研发、试点实施、硬件采买；定期向资金方报备执行情况。

## 6. 角色与职责

| --- | --- |
| CEO | 政府关系、资源协调、重大合同审批 |

## 7. 工具与系统
- PMO：Jira / 禅道 → 需求/缺陷管理。
- 合规：Notion/Confluence → 文档 + 模板；签署 DocuSign。
- 数据安全：密码管理器（Bitwarden）、日志（ELK/Loki）。
- DevOps：GitHub/GitLab + CI + SBOM（Syft/Grype）+ 安全扫描（Semgrep）。

## 8. 后续迭代
- 每季度更新 SOP，纳入最新政策/标准（如 NMPA 指南更新）。
- 将试点经验沉淀为行业白皮书、标准提案，增强政策影响力。
